A simple and intuitive web application for creating, editing, and managing your notes.
Built with the MERN stack (MongoDB, Express, React, Node.js).
Styled with TailwindCSS & DaisyUI 

Features
- Create, edit, and delete notes
- Notes are saved in a MongoDB database
- Real-time updates after adding or editing a note
- Responsive and modern UI built with React
- RESTful API built using Express and Node.js

How to run
Backend: npm start
Frontend: npm run dev
